# Luxe AI Chat (Android)

Aplicación Android de chat con IA con enfoque profesional y diseño premium, construida con Jetpack Compose.

## Características

- Interfaz moderna con gradientes y estética premium.
- Burbujas de chat para usuario e IA.
- Animación de "typing" para respuestas de la IA.
- Auto-scroll al último mensaje.
- Arquitectura simple con `ViewModel`.

## Requisitos

- Android Studio Jellyfish o superior.
- Android SDK 34.
- JDK 17.

## Ejecución

1. Abre el proyecto en Android Studio.
2. Sincroniza Gradle.
3. Ejecuta en emulador o dispositivo físico.
